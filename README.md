# ✨ Custom-Cursor-Blend-Effect

A modern and minimal custom cursor interaction project built using HTML, CSS, and JavaScript.
This project replaces the default cursor with a smooth, animated circular element that follows mouse movement and creates a visually appealing blend effect.

## 🚀 Project Overview

A small static web project demonstrating modern front-end structure with a single-page layout:

- `index.html` — markup and entrypoint
- `style.css` — styles and responsive rules
- `script.js` — interactive behavior and DOM logic


## 🔍 Highlights / Features

- Custom circular cursor
- Smooth mouse tracking animation
- Blend mode effect using mix-blend-mode: difference
- Clean and minimal UI
- Fully responsive layout


## 🎬 Demo

To run the site locally, simply open `index.html` in any modern browser. For a nicer developer workflow, use a static server (instructions below).

## Usage

- Open the site in your browser and interact with the UI.
- Inspect `script.js` to understand the JavaScript behavior and hooks into the DOM.
- Modify `style.css` to change visual styling and responsive breakpoints.
